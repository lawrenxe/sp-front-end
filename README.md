# Sneaker Paparazzi
Developer: @lawrenxe


## Prerequisite:
- Node (v16.0 or later)
- npm (v8.0 or later)
- Download Backend Source Code from [sp-back-end](https://github.com/lawrenxe/sp-back-end)

## How to compile:
1. npm install
2. npm run dev
3. click the link in the terminal.
